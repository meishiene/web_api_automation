# Project Overview

## 项目目标
Web API自动化测试平台是一个轻量级的API测试管理工具，用于创建、管理和执行API测试用例，跟踪测试结果。

## 系统功能
- **用户管理**: 用户注册和登录认证
- **项目管理**: 创建和管理测试项目，支持项目归属和权限控制
- **测试用例管理**: 创建、编辑、删除API测试用例，支持GET/POST/PUT/DELETE等HTTP方法
- **测试执行**: 执行单个或批量测试用例，实时获取测试结果
- **测试报告**: 查看历史测试执行记录和测试结果详情

## 核心能力
- 支持HTTP/HTTPS协议的API测试
- 支持自定义请求头和请求体（JSON格式）
- 支持状态码和响应体的断言验证
- 异步测试执行，支持超时控制（30秒）
- 测试结果持久化存储，支持历史查询

## 用户角色
- **测试人员**: 系统主要用户，创建测试用例并执行测试
- **开发者**: 使用系统验证API功能和回归测试

## 系统边界
- **内部系统**:
  - 前端Vue.js应用（端口5173）
  - 后端FastAPI服务（端口8000）
  - SQLite数据库（本地文件存储）

- **外部依赖**:
  - 被测试的第三方API服务（通过httpx调用）
  - 无其他外部存储或中间件依赖

- **不包含功能**:
  - 不支持并发测试执行
  - 不支持测试调度和定时任务
  - 不支持团队协作和多用户共享
  - 不支持复杂的测试场景（如参数化、数据驱动）
