# 模块技能：用户、组织与认证鉴权

## 1. 模块目标

将当前简化认证升级为企业级身份认证与权限控制体系，支撑多用户协作与资源隔离。

## 2. 当前状态

- 阶段状态：未开始
- 关联阶段：阶段 1~2
- 风险等级：高

## 3. 范围边界

### In Scope
- JWT/Refresh Token 认证机制
- 密码哈希存储
- 角色与权限模型（RBAC）
- 项目成员权限控制
- 统一鉴权中间件/依赖

### Out of Scope
- SSO/LDAP/OAuth2 企业登录（后续集成阶段）

## 4. 关键任务

1. 设计认证令牌结构与刷新机制
2. 改造登录、注销、刷新接口
3. 建立角色权限模型与权限校验点
4. 完成项目级资源访问控制
5. 补齐鉴权失败与越权场景错误处理

## 5. 交付物

- 认证与权限模型设计文档
- 认证接口与鉴权依赖实现
- 权限矩阵清单
- 鉴权测试用例

## 6. 测试要求

- 登录/刷新/过期/登出流程测试
- 越权访问测试
- 不同角色访问矩阵测试
- 密码安全性与错误处理测试

## 7. 完成定义（DoD）

- 不再依赖 `X-User-ID` 作为正式认证方案
- 受保护资源均具备权限校验
- 核心认证流程测试通过
